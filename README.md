# 🌱 Eco Wave – AI-Based Eco-Cultural Tourism Platform

## 📌 Overview
Eco Wave is an AI-powered tourism platform designed to promote sustainable travel using intelligent recommendations and environmental tracking.

## 🚀 Features
- AI-based tourism recommendations
- Image classification using deep learning
- Eco-impact tracking system
- Database integration for tourism data

## 🧠 Technologies Used
- JavaScript (Node.js)
- HTML
- Python (for AI models)
- Machine Learning / Deep Learning
- SQL Database

## 📊 Dataset
- Telangana Tourism Dataset
- Image dataset (1200+ images used for training)

## 💡 Outcome
- Encourages eco-friendly tourism
- Supports local communities
- Provides smart travel insights
